# Harry Potter Fan Site

A responsive, interactive fan site dedicated to the **Harry Potter** universe, built using **HTML** and **Tailwind CSS**.
Explore Hogwarts, houses, books and movies from the wizarding world.

## Features
  
- **Hogwarts Information**: Sections for Hogwarts, houses, books, and movies.  
- **Interactive Cards**: Hover effects on house, book, and movie cards for a polished look.  
- **Smooth Scrolling**: Scroll between sections with a smooth animation.
