# Harry Potter Fan Site

A responsive, interactive fan site dedicated to the **Harry Potter** universe, built using **HTML**, **Tailwind CSS**, and **JavaScript**. Explore Hogwarts, houses, books, movies, and enjoy background music from the wizarding world.

## Features

- **Mobile Menu**: Hamburger menu with smooth toggle functionality for mobile screens.  
- **Background Music**: Play/pause Hedwig's Theme with a single click.  
- **Hogwarts Information**: Sections for Hogwarts, houses, books, and movies.  
- **Interactive Cards**: Hover effects on house, book, and movie cards for a polished look.  
- **Smooth Scrolling**: Scroll between sections with a smooth animation.
